**文件/目录检查项配置**:
- **检查项描述** (必填)
  - 文本输入框
  - 简要描述检查项用途
- **文件/目录路径** (必填)
  - 文本输入框
  - 功能简述：指定要检查的文件或目录路径是否存在
  - 输入示例：/etc/hosts、/var/log/
  - 输出示例：
    - 采集的文件或目录的路径：/etc/hosts、/var/log/
    - 比较结果：
      - 匹配
      - 不匹配
        - 不存在的文件或目录路径
  - 支持文件路径或目录路径
  - 错误处理：
    - 路径不存在时，提示错误
    - 路径不是文件或目录时，提示错误
    - 路径是目录、文件，但没有读取权限时，提示错误
- **文件时间** (可选)
  - 功能简述：指定要检查的文件或目录的时间范围是否跟基准文件匹配或在指定范围内
  - 输入示例：2021-05-01 00:00:00
  - 输出示例：
    - 采集的文件或目录的时间：2021-05-01 00:00:00
    - 比较结果：
      - 匹配
      - 不匹配
        - 不匹配的基线时间范围和采集的时间范围
        - 采集的文件或目录的时间超出了指定范围
        - 采集的文件或目录的时间低于指定下限值
        - 采集的文件或目录的时间高于指定上限值
  - 支持秒级时间戳
  - 比较基准选择:
    - 比较基准 (与快照比较)
    - 指定时间范围
  - 范围设置:
    - 下限值 (秒)
    - 上限值 (秒)
  - 错误处理：
    - 路径不存在时，提示错误
    - 路径不是文件或目录时，提示错误
    - 路径是目录、文件，但没有读取权限时，提示错误
    - 指定时间范围时，下限值不能大于上限值，上限值不能小于下限值，下限不能小于0
- **文件大小** (可选)
  - 功能简述：指定要检查的文件或目录的大小范围
  - 输入示例：1024000（1MB）、104857600（100MB）
  - 输出示例：
    - 采集的文件或目录的大小：1024000（1MB）、104857600（100MB）
    - 比较结果：
      - 匹配
      - 不匹配
        - 不匹配的基线大小范围和采集的大小范围
        - 采集的文件或目录的大小超出了指定范围
        - 采集的文件或目录的大小低于指定下限值
        - 采集的文件或目录的大小高于指定上限值
  - 支持字节、KB、MB、GB 等单位
  - 基准选择:
    - 比较基准
    - 指定大小
  - 范围设置:
    - 下限值 (字节)
    - 上限值 (字节)
  - 错误处理：
    - 路径不存在时，提示错误
    - 路径不是文件或目录时，提示错误
    - 路径是目录、文件，但没有读取权限时，提示错误
    - 指定大小范围时，下限值不能大于上限值，上限值不能小于下限值，下限不能小于0
    
- **属主** (可选)
  - 基准选择:
    - 比较基准
    - 指定用户
  - 用户输入框
  - 功能简述：指定要检查的文件或目录的属主
  - 输入示例：root、user123
  - 输出示例：
    - 采集的文件或目录的属主：root、user123
    - 比较结果：
      - 匹配
      - 不匹配
        - 不匹配的基线属主信息和采集的属主信息      
  - 支持用户名或 UID
  - 错误处理：
    - 指定用户不存在时，提示错误
- **属组** (可选)
  - 基准选择:
    - 比较基准
    - 指定组
  - 组输入框
  - 功能简述：指定要检查的文件或目录的属组
  - 输入示例：root、group123
  - 输出示例：
    - 采集的文件或目录的属组：root、group123
    - 比较结果：
      - 匹配
      - 不匹配
        - 不匹配的基线属组信息和采集的属组信息
  - 支持组名或 GID
  - 错误处理：
    - 指定组不存在时，提示错误
- **文件权限** (可选)
  - 基准选择:
    - 比较基准
    - 指定权限
  - 权限输入框 (八进制，如 644、755)
  - 功能简述：指定要检查的文件或目录的权限
  - 支持八进制权限表示法
  - 输入示例：644、755
  - 输出示例：
    - 采集的文件或目录的权限：644、755
    - 比较结果：
      - 匹配
      - 不匹配
        - 不匹配的基线权限信息和采集的权限信息
  - 错误处理：
    - 指定权限格式错误时，提示错误
- **MD5 码** (可选)
  - 基准选择:
    - 比较基准
    - 指定 MD5 值
  - MD5 值输入框
  - 功能简述：指定要检查的文件或目录的 MD5 码
  - 输入示例：1234567890abcdef1234567890abcdef
  - 输出示例：
    - 采集的文件或目录的 MD5 码：1234567890abcdef1234567890abcdef
    - 比较结果：
      - 匹配
      - 不匹配
        - 不匹配的基线 MD5 值和采集的 MD5 值
  - 支持 32 位 MD5 值
  - 错误处理：
    - 指定 MD5 值格式错误时，提示错误

**文件内容检查项配置**:
- **检查项描述** (必填)
- **文件路径** (必填)
- **文件类型** (必填)
  - 普通文本文件
  - 内核参数文件
- **普通文件文本比较模式** (当文件类型为普通文本文件时显示):
  - 与比较基准中相同普通文本文件的内容进行比较
  - 指定普通文本部分文件内容进行比较
  - 包含指定文本：是/否
  - 不包含指定文本：是/否
  - 功能简述：指定要检查的普通文本文件的内容
  - 输入示例：/etc/hosts、/var/log/syslog
  - 输出示例：
    - 与比较基准中相同普通文本文件的内容进行比较：
      - 采集普通文本文件内容
    - 指定普通文本部分文件内容进行比较：
      - 采集普通文本部分文件内容
    - 比对结果：
      - 普通文本内容是否与基准普通文本内容匹配
        - 匹配：是
        - 不匹配：否
          - 不匹配的普通文本内容
  - 错误处理：
    - 指定普通文本文件内容进行比较时，路径不存在时，提示错误
    - 指定普通文本文件内容进行比较时，路径不是文件时，提示错误
    - 指定普通文本文件内容进行比较时，路径是目录、文件，但没有读取权限时，提示错误

- **内核参数比较模式** (当文件类型为内核参数文件时显示):
  - 与比较基准中相同内核参数文件的内容进行比较
  - 与指定的值进行比较
  - 内核参数值输入框
  - 功能简述：指定要检查的内核参数文件的内容
  - 输入示例：net.ipv4.ip_forward=1或/proc/sys/net/ipv4/ip_forward
  - 输出示例：
   - 与比较基准中相同内核参数文件的内容进行比较：
     - 采集内核参数文件内容
   - 与指定的值进行比较：
     - 采集指定内核参数值
   - 比对结果：
     - 内核参数值是否与基准值匹配
       - 匹配：是
       - 不匹配：否
         - 不匹配的内核参数值内容
  - 支持正则表达式
  - 错误处理：
    - 指定内核参数格式错误时，提示错误
    - 指定内核参数值为空时，提示错误

**路由表检查项配置**:
- **检查项描述** (必填)
- **路由表模式** (必填)
  - 采集当前系统的全量路由表信息
  - 检查是否包含指定的路由规则
- **路由规则** (当路由表模式为检查路由规则时显示)
  - 路由规则输入框
  - 功能简述：指定要检查的路由规则
  - 输入示例：192.168.1.0/24 via 192.168.1.1 dev eth0 
  - 输出示例：
    - 采集路由表信息：
      - 目标主机路由表信息内容
    - 检查路由规则：
      - 包含指定路由规则：是/否
    - 比对结果：
      - 路由表信息是否与基准路由表信息匹配
        - 匹配：是
        - 不匹配：否
          - 不匹配的路由规则内容
  - 错误处理：
    - 指定路由规则格式错误时，提示错误
    - 采集路由表信息失败时，提示错误
    - 指定路由规则为空时，提示错误
